# Code style

-   Utilise la librairie phospor pour les icones, elle est installée sur le projet.
-   Essaye de faire du code propre, lisible, otpimisé et compréhensible. Quand un composant devient trop gros, n'hésite pas à le splitter en plusieurs composant.
-   Ne fait pas de duplication de code, si une fonction est utilisé à plusieurs endroits alors extrait la dans un fichier.
-   Fait tout le css en tailwind
-   Prend aussi en compte l'aspect de la performance de l'app.
-   Pour le design de ce que tu crée, prend toujours en compte le design déjà présent sur l'app
